void application::run ( )

application run

This function is the application that makes the image change when you use the sensor.
You can change the threshold of every axis via the application constructor.
The constructor has base values and only needs the sensor and oled display if you choose to keep the threshold base values.
The posThreshX and Y are the values to determine when the acceleration or tilt in the right or up direction is high enough for the display to show the next image.
The negThreshX and Y are the values to determine when the acceleration or tilt in the left or down direction is high enough for the display to show the next image.
The high and low reset values are the values where the data from the axes has to be in between in order to change to image again.
As long as the axes have not been in the reset area at the same time the image will not change, no matter how far you tilt or how fast you accelerate the sensor.
If the negThresh value is bigger than the posThresh value, they wil automatically be swapped. The same happens with the reset values.
